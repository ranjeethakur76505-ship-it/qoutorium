# 🏛️ Quoterium

Quoterium is an elegant, highly responsive Single Page Application (SPA) that curates timeless wisdom across four distinct spaces: Philosophy, Psychology, Science, and Poetry. 

## ✨ Features
- **Seamless SPA Navigation:** Switch themes instantly with zero browser reloads.
- **Dynamic Chromatic Themes:** The entire UI beautifully morphs color schemes based on the selected category.
- **Interactive Controls:** Features shuffle animations, history navigation, quote copying, and bookmarks.
- **Responsive Layout:** Optimized with clean editorial typography matching desktop, tablet, and mobile displays.

## 🛠️ Built With
- Semantic HTML5
- Custom CSS3 Properties (Variables & Transitions)
- Vanilla JavaScript
